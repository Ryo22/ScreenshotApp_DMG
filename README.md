# 📷 ScreenshotApp (DMG版)

macOS 用のウィンドウ指定・領域指定スクリーンショットアプリです。  
このリポジトリでは、ソースコードを隠蔽し、簡単にインストール・実行できる **DMG ファイル** を配布しています。

## 📥 インストール方法

1. **DMG ファイルをダウンロード**
   - このリポジトリにある [ScreenshotApp.dmg](./ScreenshotApp.dmg) をクリックし、`Download` ボタンからダウンロードしてください。

2. **アプリケーションフォルダへ移動**
   - ダウンロードした `.dmg` ファイルを開き、`ScreenshotApp.app` を `Applications` フォルダへドラッグ＆ドロップしてください。

3. **アプリの起動**
   - `Applications` フォルダから `ScreenshotApp.app` を開いてください。
   - ※ 開発元未確認の警告が出る場合は、`システム設定` → `プライバシーとセキュリティ` から「このまま開く」を許可してください。

---

## ⚙️ macOS の各種権限設定

アプリを正常に動作させるために、以下の権限を許可する必要があります。

1. **画面収録** の許可
   - `システム設定` → `プライバシーとセキュリティ` → `画面収録`
   - `ScreenshotApp` を **ON** にする

2. **アクセシビリティ** の許可 (自動タップ機能用)
   - `システム設定` → `プライバシーとセキュリティ` → `アクセシビリティ`
   - `ScreenshotApp` を **ON** にする

---

## 🚀 主な機能

- **ウィンドウ指定キャプチャ**: 特定のウィンドウが他のアプリに隠れていても、その内容だけを撮影できます。
- **領域していキャプチャ**: 画面上の任意の範囲をドラッグで指定して継続的に撮影できます。
- **自動タップ機能**: 指定した間隔で自動クリックしながら撮影を繰り返せます。
- **プレビュー表示**: アプリ画面の下部で、直前に撮影した内容をリアルタイムに確認できます。
- **キーボードショートカット**: `Cmd + Ctrl + S` でキャプチャの開始/停止を素早く行えます。
